About Lighthouse

Lighthouse is an open-source, automated tool for improving the quality of web pages. It audits web pages for performance, accessibility, progressive web apps, SEO, and more.

What does Lighthouse test for?

  • Performance: Measures how fast your page loads and how quickly users can interact with it
  • Accessibility: Checks for common accessibility issues and ensures your site is usable by everyone
  • Best Practices: Validates modern web development best practices and security
  • SEO: Ensures your site follows search engine optimization best practices

Score Interpretation

  • 90-100 (Good): The page follows best practices and performs well
  • 50-89 (Needs Improvement): The page has room for improvement
  • 0-49 (Poor): The page needs significant improvements
